![GöBorg Logo](assets/GöBorg.png)  
GöBorg 0.0.0.1

## Instalation:
### Requirements:
 - podman or docker
 - git

### Instalation process:
 - clone the repository with `git clone https://github.com/nonelone/go-borg`
 - cd to it using `cd go-borg`
 - run `run.sh` or `run.bat`
